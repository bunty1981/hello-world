# AI Copilot Vibe App

A calendar sharing application that allows multiple users to share their personal calendars in view-only and view-and-edit modes.

## Setup

1. Install dependencies: `npm run install-all`
2. Start the application: `npm start`

This will start the backend on port 5000 and frontend on port 3000.

## Development

- Frontend: React app in `client/`
- Backend: Node.js Express in `server/`

## Testing

Use GitHub Codespaces for easy setup with the provided `.devcontainer` configuration.
